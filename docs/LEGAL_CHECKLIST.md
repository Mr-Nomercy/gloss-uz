# Legal & Compliance Checklist

## Pre-Launch Requirements
- [ ] Register company in Uzbekistan (IT Park resident recommended)
- [ ] Obtain payment system license (or agency agreement with Click/Payme)
- [ ] Register as personal data operator with Uzbekistan authorities
- [ ] Draft Terms of Service (Uzbek, Russian, English)
- [ ] Draft Privacy Policy (Uzbek, Russian, English)
- [ ] Sign DPAs with all third-party processors
- [ ] Conduct Data Protection Impact Assessment (DPIA)
- [ ] Implement age verification mechanism
- [ ] Register trademark (Gloss name + logo)
- [ ] Obtain liability insurance (min $1M coverage)

## Operational Compliance
- [ ] Data Protection Officer appointed (name + contact published)
- [ ] Breach notification procedure documented and tested
- [ ] AML/CFT reporting procedure documented
- [ ] Transaction monitoring system operational
- [ ] Regular KYC re-verification (annual for high-volume sellers)
- [ ] Content moderation policy published
- [ ] Transparency report published annually

## Ongoing Obligations
- [ ] Monthly dependency security scans (Snyk/Dependabot)
- [ ] Quarterly vulnerability assessments
- [ ] Annual penetration testing
- [ ] Annual ISO 27001 or equivalent certification review
- [ ] Biannual DPIA review
- [ ] Continuous audit log monitoring
